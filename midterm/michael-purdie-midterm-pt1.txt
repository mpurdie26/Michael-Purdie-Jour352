2.  Describe the purpose of a DIV tag.

 A <div> tag is used to separate different sections of code that will have the same CSS style rules applied all the code within the <div> tag. An ID or class can be assigned to a <div>, in which case it can by called upon in the CSS file to apply rules to that specific tag. Also, I <div> tag can make your code more readable.
